This file is used as the workbench to create the generated buttons templating code

The button preset consists of a pipe separated multiple values string.
Each value is dedicated for a specific button option:



1 WIDTH
2 HEIGHT
3 BG_COLOR
4 BG_IMAGE
5 BORDER_COLOR
6 BORDER_RADIUS
7 FONT_COLOR
8 FONT_SIZE
9 TEXT_POSITION
10 ICON_IMAGE
11 ICON_IMAGE_WIDTH
12 ICON_POSITION
13 PADDING_TOP
14 PADDING_RIGHT
15 PADDING_BOTTOM
16 PADDING_LEFT
17 FONT_FAMILY
18 FONT_CSS

example value for component: DEFAULT_BUTTON_PRESET

100%|38px|99c739|https://imgur.com/clZqdfM.gif|7ea613|3px|ffffff|13px|center|http://via.placeholder.com/16x16|16px|right|null|10px|null|10px|Arial,sans-serif|text-decoration:underline
empty value must be 'null' in order to make this work also selligent version must be 6.3.4+ 
REGEX:

to base from DEFAULT_BUTTON_PRESET
GETPROP\(DYNCONTENT\.DEFAULT_BUTTON_PRESET,([0-9]*?),'\|'\)
TO
{\1}

to base from ITEM.BUTTON_PRESET
GETPROP\(ITEM\.BUTTON_PRESET,([0-9]*?),'\|'\)
TO
{\1}

default button replace:
(\{)([0-9]*?)(\})
TO
GETPROP(DYNCONTENT.DEFAULT_BUTTON_PRESET,\2,'|')

ITEM button preset replace:
(\{)([0-9]*?)(\})
TO
GETPROP(ITEM.BUTTON_PRESET,\2,'|')
